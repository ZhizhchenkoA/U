# Турнир Угольникова

## Описание проекта

**«Турнир Угольникова»** - это пошаговая стратегическая игра на графе, реализованная на языке C++ с использованием библиотеки **Qt 6** для графического интерфейса.  
Игровое поле представляет собой карту субъектов Российской Федерации, загружаемую из GeoJSON-файлов. Игрок и компьютер по очереди перемещаются между соседними регионами, стремясь первым достичь конечного региона или лишить соперника возможных ходов.

Проект разработан в учебных целях и сочетает в себе алгоритмы на графах, пользовательский интерфейс и работу с реальными географическими данными.

---

## Основные возможности

- Загрузка карты России из JSON и GeoJSON файлов  
- Представление карты в виде графа регионов  
- Визуализация карты с подсветкой посещённых регионов  
- Пошаговая игра "человек vs компьютер"
- Стратегия компьютера на основе BFS и расстояний в графе  
- Обновление карты в реальном времени при ходе игрока  

---

## Архитектура проекта

Проект разделён на несколько логических уровней:

### Логика игры
- `Game` — игровая логика, правила, ходы, проверка победы
- `Queue` — очередь для BFS
- `List<T>` — собственная реализация связного списка
- `String` — собственная строка

### Модель данных
- `AbstractSubject` — регион (вершина графа)
- `SubjectRussia` — конкретный субъект РФ
- `Map` — карта регионов и их соседств

### Графический интерфейс
- `MapWidget` — отрисовка карты регионов
- `PlayerWindow` — основное игровое окно

### Связующий слой
- `Presenter` — связывает `PlayerWindow` и `MapWidget`, обрабатывает сигналы и обновляет карту

## Сборка и запуск

### Требования

- Linux
- GCC / Clang с поддержкой C++17
- Qt 6
- CMake ≥ 3.16

### Сборка

```bash
mkdir build
cd build
cmake ..
make
```

### Запуск
```
./Tournament
```
